# Zuma Deluxe Highscore Reader
An highscore file reader for the beloved sphere-matching game Zuma Deluxe

# Build
## Dependicies
- GNU C++11
- Makefile

## Building steps
```bash
cd zuma-highscore-reader
make                 # for regular bulid
make DGLAGS=-DDEBUG  # for debugging output
```

## Execution
- Linux: `./bin/main`
- Windows: `bin\main`
